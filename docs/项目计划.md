## 一、项目总览

### 1.1 项目目标

构建一个个人使用的AI量化交易平台，覆盖**A股**和**期货**两大市场，实现：
- ✅ 自动化数据采集与清洗
- ✅ 多策略回测验证与可视化
- ✅ 期货CTA回测引擎（保证金/多空/逐日盯市）
- ✅ 期货实盘连接（C++桥接DLL + SimNow 7x24）
- ⬜ AI因子挖掘与预测
- ⬜ 全自动实盘交易与运维

### 1.2 开发周期

| 阶段 | 时间 | 主要目标 | 状态 |
|------|------|----------|------|
| **Phase 1** | 已完成 | 基础框架 + 数据采集 | ✅ 已完成 |
| **Phase 2** | 已完成 | 回测引擎 + 策略开发 | ✅ 已完成 |
| **Phase 3** | 2-3周 | 期货CTA系统（回测+实盘引擎+合约管理+风控升级+CTP实盘连接） | 🔄 进行中 (90%) |
| **Phase 4** | 3-4周 | AI模块 + 模型训练 | ⬜ 未开始 |
| **Phase 5** | 2-3周 | 实盘对接 + 运维体系 | ⬜ 未开始 |
| **Phase 6** | 持续 | 优化迭代 + 扩展功能 | ⬜ 未开始 |

### 1.3 优先级定义

| 优先级 | 说明 | 示例 |
|--------|------|------|
| **P0** | 核心功能，必须完成 | 数据采集、回测引擎、CTP接口、实盘引擎、合约管理、风险控制、告警通知 |
| **P1** | 重要功能，近期完成 | 参数优化、多周期策略、投资组合管理、Web监控面板 |
| **P2** | 增强功能，后续完成 | AI因子挖掘、机器学习模型、定时任务调度 |

### 🔴 P0 待完成 (实盘交易核心)

#### 1️⃣ 实盘策略引擎 (`src/engine/live_engine.py`) — 最高优先级

回测与实盘的核心差异：

| 回测 | 实盘 |
|------|------|
| 历史数据回放 | 实时Tick/Bar推送 |
| 虚拟资金 | 真实账户 |
| 无网络延迟 | 需要考虑网络延迟 |
| 固定滑点 | 真实成交价 |
| 无撤单概念 | 需要撤单重发逻辑 |

**需要实现的功能：**
- [x] 实时行情订阅（Tick/K线）
- [x] 策略信号生成与执行
- [x] 异步订单执行（下单→等待成交→确认）
- [x] 撤单重发机制（超时未成交自动撤单重发）
- [x] 日内时间控制（集合竞价/开盘/收盘特殊处理）
- [x] 状态持久化（重启恢复持仓/订单状态）

#### 2️⃣ 合约管理与换月移仓 (`src/trade/contract_manager.py`) — 期货特有

期货与A股最大区别：**合约有到期日，必须换月！**

**需要实现的功能：**
- [x] 品种规格数据库（26个品种的乘数/保证金/最小变动价位）
- [x] 合约代码解析与生成
- [x] 主力合约跟踪与换月检测
- [x] 换月移仓执行（平旧仓→开新仓）
- [x] 交割月限制（临近交割月提高保证金+限制开仓）
- [x] 合约管理器与实盘引擎集成（主循环中周期性检查）
- [ ] 持仓量数据源实时接入（需要CTP行情数据）
- [ ] 合约参数管理（乘数/保证金率/最小变动价位）
- [ ] 交割月限制（临近交割月提高保证金/限制开仓）

#### 3️⃣ 实时风控与保证金监控 (`src/trade/risk_manager.py` 升级)

当前已有基础风控，需补充：

- [x] **保证金实时计算**: 通过 ContractManager 获取乘数+保证金率，动态计算
- [x] **风险度监控**: 风险度 = 占用保证金/总权益，80%警戒/90%危险/100%强平
- [x] **盘中动态风控**: 浮动盈亏实时影响总权益和风险度
- [x] **涨跌停板保护**: 涨停不能开空/跌停不能开多
- [x] **强平预警**: LiquidationWarningRule 周期性检查并告警
- [x] **RiskManager 升级**: calc_margin/calc_total_margin/calc_risk_ratio/get_margin_status
- [ ] 自动减仓预案（达到强平线时自动部分平仓）
- [ ] **涨跌停板保护**: 涨停不能开空/跌停不能开多
- [ ] **大额报撤单监控**: 交易所风控规则（自成交防范）
- [ ] **强平预警**: 低于预设风险度阈值自动通知

|------|------|------|----------|
| 隔夜跳空 | 中 | 高 | 控制隔夜仓位、止损单 |
| 保证金不足强平 | 中 | 极高 | 实时风险度监控、自动预警 |
| 流动性枯竭 | 低 | 高 | 持仓量过滤、剔除僵尸合约 |
| 交割月临近 | 中 | 中 | 提前换月、避免进入交割月 |
| 系统故障 | 中 | 高 | 监控告警、断线自动重连、状态持久化 |

### 9.3 业务风险

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 策略失效 | 高 | 高 | 持续研究、多策略分散、定期评估 |
| 市场极端行情 | 低 | 高 | 风控熔断机制、硬止损 |
| 资金管理不当 | 中 | 高 | 凯利公式、固定比例仓位、风险预算 |
