## 一、项目总览

### 1.1 项目目标

构建一个个人使用的AI量化交易平台，实现：
- ✅ 自动化数据采集
- ✅ 策略回测验证
- ⬜ AI因子挖掘与预测
- ⬜ 实盘自动交易

### 1.2 开发周期

| 阶段 | 时间 | 主要目标 | 状态 |
|------|------|----------|------|


| **Phase 1** | 2-3周 | 基础框架 + 数据采集 | 🔄 进行中 (60%) |
| **Phase 2** | 2-3周 | 回测引擎 + 策略开发 | 🔄 进行中 (80%) |
| **Phase 3** | 3-4周 | AI模块 + 模型训练 | ⬜ 未开始 |
| **Phase 4** | 2-3周 | 实盘接口 + 风控系统 | ⬜ 未开始 |
| **Phase 5** | 持续 | 优化迭代 + 扩展功能 | ⬜ 未开始 |

### 1.3 优先级定义

| 优先级 | 说明 | 示例 |
|--------|------|------|
| **P0** | 核心功能，必须完成 | 数据采集、回测引擎 |
| **P1** | 重要功能，近期完成 | 技术指标、可视化 |
| **P2** | 增强功能，后续完成 | 更多策略、优化功能 |
| **P3** | 可选功能，有空再做 | 高级特性、扩展接口 |
