## A股回测

### 1. 采集A股数据

```bash
python scripts/collect_data.py
```

采集结果存储在 `data/raw/stocks/` 目录下的Parquet文件。

### 2. 一键运行回测 + 生成报告

```bash
python scripts/run_backtest.py
```

输出：
- **控制台**: 5个策略的绩效对比表格
- **reports/**: 1个多策略对比 + 5个策略独立HTML报告

### 3. 查看报告

```bash
# 直接在浏览器中打开
start reports/strategy_comparison.html
start reports/backtest_双均线\(10,30\)_*.html
```

### 4. A股回测代码示例

```python
from src.backtest.engine import BacktestEngine
from src.strategy.trend_strategy import SmaCrossStrategy

# 加载数据
df = pd.read_parquet("data/raw/stocks/000001.parquet")
df = df.set_index('date')

# 创建回测引擎
engine = BacktestEngine(
    initial_cash=100000,
    commission=0.0003,
    slippage=0.0001,
    stamp_duty=0.001,
)

# 运行策略
strategy = SmaCrossStrategy(params={
    'fast_period': 10,
    'slow_period': 30
})
result = engine.run(df, strategy, symbol="000001")

# 查看结果
engine.print_result(result)
```

### 5. 可用A股策略

| 策略 | 类名 | 参数 |
|------|------|------|
| 双均线 | SmaCrossStrategy | fast_period=10, slow_period=30 |
| MACD | MACDStrategy | fast=12, slow=26, signal=9 |
| 布林带 | BollingerBandsStrategy | period=20, std_dev=2 |
| RSI超买超卖 | RSIStrategy | period=14, overbought=70, oversold=30 |
| RSI2短线反转 | RSI2Strategy | period=2, threshold=10 |

| 原油 | SC | 1000桶/手 | 上期能源 |
| 沪深300股指 | IF | 300元/点 | 中金所 |

### 2. 期货回测代码示例

```python
from src.backtest.futures_engine import FuturesBacktestEngine
from src.strategy.futures_strategy import DualMaCrossStrategy

# 加载期货数据
data = pd.read_parquet("data/raw/futures/CU.parquet")
data = data.set_index('date')

# 创建期货回测引擎
engine = FuturesBacktestEngine(
    initial_capital=500000,
    commission_rate=0.0001,      # 万分之一
    slippage=1,                   # 滑点1个tick
    margin_rate=0.12,            # 保证金率12%
    contract_multiplier=5,       # 合约乘数5吨/手
    price_tick=10,               # 最小变动价位10元
    big_point_value=5,           # 每点价值5元
    settle_type='future',        # 期货结算方式
)

# 运行策略
strategy = DualMaCrossStrategy(params={
    'fast_period': 10,
    'slow_period': 30,
    'atr_period': 14,
    'atr_multiplier': 2.0,
})
result = engine.run(data, strategy, symbol="CU")

# 查看结果
engine.print_result(result)
```

### 3. 可用期货策略

| 策略 | 类名 | 核心逻辑 |
|------|------|----------|
| 双均线CTA | DualMaCrossStrategy | 双均线交叉 + ATR止损 + 移动止损 + 仓位管理 |
| 趋势通道 | SimpleTrendStrategy | 唐奇安通道突破 + ATR跟踪 |

### 4. 期货回测核心特性

- **双向交易**: 支持做多/做空
- **保证金计算**: 按合约价值和保证金率计算
- **逐日盯市**: 每日按结算价浮动盈亏
- **ATR止损**: 波动率动态止损
- **移动止损**: 盈利后上移止损线
- **仓位管理**: 按总资金比例计算手数

## 策略开发

### A股策略

```python
from src.strategy.base import BaseStrategy

class MyStrategy(BaseStrategy):
    def __init__(self, params: dict = None):
        super().__init__(params)
        self.fast = self.params.get('fast_period', 10)
        self.slow = self.params.get('slow_period', 30)
    
    def generate_signals(self, data):
        close = data['close']
        fast_ma = close.rolling(self.fast).mean()
        slow_ma = close.rolling(self.slow).mean()
        
        signals = pd.Series(0, index=data.index)
        signals[(fast_ma > slow_ma) & (fast_ma.shift(1) <= slow_ma.shift(1))] = 1
        signals[(fast_ma < slow_ma) & (fast_ma.shift(1) >= slow_ma.shift(1))] = -1
        return signals
```

### 期货CTA策略

```python
from src.strategy.futures_strategy import BaseFuturesStrategy

class MyFuturesStrategy(BaseFuturesStrategy):
    def generate_signals(self, data):
        """
        返回信号DataFrame，包含:
        - direction: 1=开多, -1=开空, 0=空仓
        - stop_loss: 止损价 (可选)
        """
        close = data['close']
        fast_ma = close.rolling(10).mean()
        slow_ma = close.rolling(30).mean()
        atr = self.calculate_atr(data, 14)
        
        signals = pd.DataFrame(index=data.index, columns=['direction', 'stop_loss'])
        
        buy_signal = (fast_ma > slow_ma) & (fast_ma.shift(1) <= slow_ma.shift(1))
        sell_signal = (fast_ma < slow_ma) & (fast_ma.shift(1) >= slow_ma.shift(1))
        
        signals.loc[buy_signal, 'direction'] = 1
        signals.loc[buy_signal, 'stop_loss'] = close - 2 * atr
        signals.loc[sell_signal, 'direction'] = -1
        signals.loc[sell_signal, 'stop_loss'] = close + 2 * atr
        signals['direction'] = signals['direction'].fillna(0)
        return signals
    
    def calculate_atr(self, data, period):
        high, low, close = data['high'], data['low'], data['close']
        tr = pd.concat([
            high - low,
            (high - close.shift()).abs(),
            (low - close.shift()).abs()
        ], axis=1).max(axis=1)
        return tr.rolling(period).mean()
```

## 可视化报告

### 生成的报告文件

运行 `scripts/run_backtest.py` 后，`reports/` 目录生成：

| 文件 | 内容 |
|------|------|
| `strategy_comparison.html` | 多策略权益曲线对比 |
| `backtest_双均线(10,30)_*.html` | 双均线报告(权益曲线+K线+热力图) |
| `backtest_MACD(12,26,9)_*.html` | MACD报告 |
| `backtest_布林带(20,2)_*.html` | 布林带报告 |
| `backtest_RSI(14,30_70)_*.html` | RSI报告 |
| `backtest_RSI2短线反转_*.html` | RSI2报告 |

### 报告内容

每份独立报告包含：
1. **权益曲线** - 资金曲线 + 回撤区域 + KPI标注
2. **K线交易信号** - K线 + 买卖点位 + 指标线
3. **月度收益热力图** - 月收益率(RdYlGn色标)

多策略对比报告包含：
1. **多策略权益曲线叠加** - 归一化对比
2. **绩效指标表格** - 总收益/年化/夏普/回撤/胜率

## 常见问题

### Q: 依赖安装失败？
A: Windows缺少C编译器，用预编译版本：
```bash
pip install pandas==2.0.3 numpy==1.26.4
```

### Q: AKShare数据获取失败？
A: 安装额外依赖：
```bash
pip install py_mini_racer curl_cffi
```

### Q: 如何加A股策略？
A: 继承 `BaseStrategy`，实现 `generate_signals()`，在 `scripts/run_backtest.py` 中注册。

### Q: 如何加期货CTA策略？
A: 继承 `BaseFuturesStrategy`，实现 `generate_signals()`（返回 direction+stop_loss 的DataFrame），在 `scripts/run_futures_backtest.py` 中注册。

### Q: 回测报告在哪？
A: 运行后 `reports/*.html`，浏览器打开。

### Q: 如何查看日志？
A: `Get-Content logs/quant-trading.log -Tail 50`

### Q: 虚拟环境权限错误？
A: 管理员PowerShell执行 `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`

### Q: 重置数据和缓存？
```bash
rm -rf logs/* data/cache/*
```
