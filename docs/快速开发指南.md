### 策略开发

```python
from src.strategy.base import BaseStrategy, Signal
import pandas as pd

class MyStrategy(BaseStrategy):
    """自定义策略"""
    
    def init(self):
        """初始化策略参数"""
        self.fast_period = self.params.get('fast_period', 10)
        self.slow_period = self.params.get('slow_period', 30)
        
        # 计算指标
        self.data['sma_fast'] = self.data['close'].rolling(self.fast_period).mean()
        self.data['sma_slow'] = self.data['close'].rolling(self.slow_period).mean()
    
    def next(self, bar: pd.Series) -> Signal:
        """每个bar调用一次，返回交易信号"""
        # 获取当前位置
        idx = bar.name if bar.name is not None else 0
        if isinstance(idx, int):
            loc = idx
        else:
            loc = self.data.index.get_loc(idx)
        
        if loc < self.slow_period:
            return Signal.HOLD
        
        # 获取指标值
        fast = self.data['sma_fast'].iloc[loc]
        slow = self.data['sma_slow'].iloc[loc]
        prev_fast = self.data['sma_fast'].iloc[loc - 1]
        prev_slow = self.data['sma_slow'].iloc[loc - 1]
        
        # 金叉买入
        if prev_fast <= prev_slow and fast > slow:
            return Signal.BUY
        
        # 死叉卖出
        if prev_fast >= prev_slow and fast < slow:
            return Signal.SELL
        
        return Signal.HOLD
```

### 回测运行

```python
from src.backtest.engine import BacktestEngine
from src.strategy.trend_strategy import SmaCrossStrategy
from src.data.collector import DataCollector

# 加载数据
collector = DataCollector()
df = collector.get_stock_history("000001")
df = df.set_index('date')

# 创建回测引擎
engine = BacktestEngine(
    initial_cash=100000,    # 初始资金
    commission=0.0003,      # 手续费万三
    slippage=0.0001,        # 滑点万一
    stamp_duty=0.001,       # 印花税千一
)

# 创建策略
strategy = SmaCrossStrategy(params={
    'fast_period': 10,
    'slow_period': 30
})

# 运行回测
result = engine.run(df, strategy, symbol="000001")

# 打印结果
engine.print_result(result)
```

## 常见问题

### Q: 依赖安装失败？

A: Windows系统缺少C编译器，部分包需要预编译版本：
```bash
pip install pandas==2.0.3 numpy==1.26.4
```

### Q: AKShare数据获取失败？

A: 需要安装额外依赖：
```bash
pip install py_mini_racer curl_cffi
```

### Q: 如何添加新策略？

A: 继承 `BaseStrategy` 类，实现 `init()` 和 `next()` 方法。

### Q: 虚拟环境激活显示权限错误？

A: 以管理员身份运行PowerShell，执行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Q: 如何查看日志？

A: 日志文件在 `logs/` 目录下，默认按天轮转：
```bash
# 查看最新日志
Get-Content logs/quant-trading.log -Tail 50

# 实时查看
Get-Content logs/quant-trading.log -Tail 10 -Wait
```

### Q: 如何修改配置？

A: 编辑 `config/secrets.yaml` 文件，配置格式如下：
```yaml
data:
  raw_dir: ./data/raw
  processed_dir: ./data/processed
  cache_dir: ./data/cache

logging:
  level: DEBUG
  rotation: 10 MB
  retention: 7 days
```

### Q: 如何重置数据和日志？

```bash
make clean
# 或手动
rm -rf logs/* data/cache/*
```
