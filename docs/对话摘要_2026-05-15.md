# 对话摘要 (压缩版)

> **生成日期**: 2026-05-15 | **项目**: AI量化交易平台

## 一、项目全貌

### 目标
构建个人AI量化交易平台，整合Qlib(AI)、Backtrader(回测)、VeighNa(实盘)的优点。

### 技术栈
| 模块 | 技术 |
|------|------|
| 语言 | Python 3.11 |
| 数据 | Pandas, NumPy, AKShare → Parquet |
| 可视化 | Plotly (Graph Objects, Subplots, Heatmap) |
| 日志 | Loguru |
| 回测引擎 | 自研事件驱动 |
| 策略框架 | 抽象基类(BaseStrategy) |
| 配置 | YAML配置 (get_config) |

## 四、核心模块详解

### 4.1 数据采集 (`src/data/collector.py`)

- **主源**: 腾讯证券 `stock_zh_a_hist_tx`
- **备用源**: 新浪财经 `stock_zh_a_daily`
- **特性**: CircuitBreaker熔断器 + RateLimiter限速 + 指数退避重试 + 本地Parquet缓存
- **缓存策略**: 最后更新≤1天直接读缓存

### 4.2 回测引擎 (`src/backtest/engine.py`)

```python
class BacktestEngine:
    # 参数: initial_cash=10万, commission=万三, slippage=万一, stamp_duty=千一(卖出)
    # run(data, strategy) → BacktestResult
    # 交易逻辑: 按手(100股), 留5%现金, 最低5元佣金
    # 绩效: 总/年化收益率, 最大回撤, 夏普比率, 胜率
```

### 4.3 策略 (`src/strategy/`)

| 策略 | 文件 | 信号逻辑 |
|------|------|----------|
| SmaCross(10,30) | trend_strategy.py | 快线上穿慢线→买, 下穿→卖 |
| MACD(12,26,9) | trend_strategy.py | MACD上穿信号线→买, 下穿→卖 |
| 布林带(20,2) | mean_reversion_strategy.py | 跌破下轨→买, 回中轨→卖, 止损5% |
| RSI(14,30/70) | mean_reversion_strategy.py | RSI<30→买, >50→卖, 止损5% |
| RSI2短线反转 | mean_reversion_strategy.py | RSI(2)<10→买(需在200日均线上方), >90→卖, 止损3% |

### 4.4 可视化 (`src/backtest/visualizer.py`)

| 函数 | 用途 |
|------|------|
| `plot_equity_curve()` | 资金曲线 + 回撤子图 + KPI注释 |
| `plot_trade_signals()` | 蜡烛图 + 买卖标记 + 指标叠加 |
| `plot_strategy_comparison()` | 多策略资金曲线对比 |
| `plot_drawdown_heatmap()` | 月度收益热力图 (RdYlGn) |
| `generate_report()` | 合并以上为单个HTML + KPI卡片 |

### 4.5 交易模块 (`src/trade/`)
- CTP Gateway + OrderManager + PositionManager + RiskManager
- 风控: 持仓限制、仓位比例、每日亏损、止损线、交易频率

## 五、Bug修复记录

| Bug | 原因 | 修复 |
|-----|------|------|
| `NameError: name 'v'` | 列表推导引用未定义变量 | 改用 `f"{heat_vals[r][c]:.1f}%"` |
| `FileNotFoundError` (路径) | 策略名含`/`被当目录分隔符 | 添加 `safe_name = name.replace("/","_")` |
| `UnicodeEncodeError` (emoji) | Windows GBK不支持📁 | 移除print中的emoji |
