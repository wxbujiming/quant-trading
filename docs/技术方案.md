## 一、项目概述

### 1.1 项目背景

本项目旨在构建一个个人使用的AI量化交易平台，整合以下开源项目的优点：
- **Qlib** (微软): AI因子挖掘、模型训练
- **VeighNa** (vn.py): 实盘交易、券商接口
- **Backtrader**: 回测框架设计思路

### 1.2 目标市场

| 市场 | 交易频率 | 策略类型 | 优先级 |
|------|----------|----------|--------|
| A股 | 日线级别 | 因子选股、AI预测 | ⭐⭐⭐⭐⭐ |
| 期货 | 日内/分钟级 | CTA、套利 | ⭐⭐⭐⭐ |
| 美股 | 日线/高频 | 趋势跟踪 | ⭐⭐⭐ |

### 1.3 资源约束

- **开发环境**: 个人电脑 (本地开发、AI训练、回测)
- **服务器**: 阿里云 2C2G (数据采集、API服务)
- **部署方式**: 本地部署为主

## 四、数据流设计

### 4.1 数据采集流程

```
AKShare/Tushare API
        ↓
    原始数据 (JSON/DataFrame)
        ↓
    数据清洗 (缺失值、异常值)
        ↓
    数据标准化 (列名、格式)
        ↓
    Parquet存储 (本地文件)
        ↓
    数据加载 (回测/交易)
```

### 4.2 策略回测流程

```
加载历史数据
        ↓
    初始化策略
        ↓
    遍历每个bar
        ↓
    生成交易信号
        ↓
    模拟订单执行
        ↓
    更新持仓/资金
        ↓
    计算绩效指标
        ↓
    生成回测报告
```

### 4.3 实盘交易流程

```
实时行情接收
        ↓
    信号生成 (策略/AI)
        ↓
    风控检查
        ↓
    订单生成
        ↓
    券商接口下单
        ↓
    订单状态监控
        ↓
    持仓更新
        ↓
    通知推送
```

### 11.3 SQLite数据库 (可选)

```sql
-- 交易记录表
CREATE TABLE trades (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    trade_date DATE NOT NULL,
    symbol VARCHAR(10) NOT NULL,
    action VARCHAR(4) NOT NULL,  -- BUY/SELL
    price DECIMAL(10, 2) NOT NULL,
    quantity INTEGER NOT NULL,
    amount DECIMAL(15, 2),
    commission DECIMAL(10, 4),
    profit DECIMAL(15, 2),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 持仓表
CREATE TABLE positions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol VARCHAR(10) NOT NULL UNIQUE,
    quantity INTEGER NOT NULL DEFAULT 0,
    avg_price DECIMAL(10, 2),
    current_price DECIMAL(10, 2),
    market_value DECIMAL(15, 2),
    profit_loss DECIMAL(15, 2),
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 策略配置表
CREATE TABLE strategies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(50) NOT NULL,
    params JSON,
    enabled BOOLEAN DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 信号记录表
CREATE TABLE signals (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    signal_date DATE NOT NULL,
    symbol VARCHAR(10) NOT NULL,
    strategy VARCHAR(50) NOT NULL,
    signal INT NOT NULL,  -- 1=BUY, -1=SELL, 0=HOLD
    confidence DECIMAL(5, 4),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_trades_date ON trades(trade_date);
CREATE INDEX idx_trades_symbol ON trades(symbol);
CREATE INDEX idx_signals_date ON signals(signal_date);
```
